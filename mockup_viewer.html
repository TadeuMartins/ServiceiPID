🔎 P&ID Digitalizer - Nova Funcionalidade: Chatbot Inteligente

✨ Funcionalidades Implementadas:

  • Descrição Automática do Processo: Após análise ou geração de P&ID, a IA gera automaticamente uma descrição técnica completa
  • Base de Conhecimento: Todos os dados são armazenados para consultas futuras
  • Chatbot Minimizável: Assistente conversacional que responde perguntas específicas sobre o P&ID
  • Integração Automática: Funciona tanto com análise de PDF quanto com geração por prompt

🎯 Recursos Principais:

  • 💬 Chatbot Minimizável: Pode ser minimizado para não ocupar espaço quando não estiver em uso
  • 📝 Descrição Automática: Gerada pela IA incluindo objetivo, etapas, equipamentos, instrumentação e segurança
  • 📜 Histórico de Conversação: Mantém registro de todas as perguntas e respostas
  • 💡 Perguntas Sugeridas: Botões de exemplo para perguntas comuns
  • 🔍 Busca Contextual: Respostas baseadas especificamente no P&ID analisado/gerado
  • 🗑️ Limpar Histórico: Opção para limpar a conversação

🚀 Novos Endpoints API:

  • POST /describe - Gera descrição completa do processo
  • POST /chat - Responde perguntas sobre o P&ID
  • POST /store - Armazena P&ID na base de conhecimento
  • GET /knowledge-base - Lista todos os P&IDs armazenados
